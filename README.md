
Сервис объявлений купли/продажи на FastAPI с аутентификацией, ролевой моделью (user/admin), пагинацией и полнотекстовым поиском.


## Права доступа

### Неавторизованные пользователи (без токена)
- `POST /user` - регистрация
- `GET /user/{id}` - просмотр пользователя
- `GET /advertisement/{id}` - просмотр объявления
- `GET /advertisement/` - поиск объявлений

### Авторизованные пользователи (группа user)
- Все права неавторизованного пользователя
- `PATCH /user/{id}` - обновление своих данных
- `DELETE /user/{id}` - удаление себя
- `POST /advertisement` - создание объявления
- `PATCH /advertisement/{id}` - обновление своего объявления
- `DELETE /advertisement/{id}` - удаление своего объявления

### Администраторы (группа admin)
- Полный доступ ко всем эндпоинтам и данным

## API Endpoints

### Аутентификация
| Метод | Эндпоинт | Описание | Доступ |
|-------|----------|----------|--------|
| POST | `/login` | Получение токена | Все |

### Пользователи
| Метод | Эндпоинт | Описание | Доступ |
|-------|----------|----------|--------|
| POST | `/user` | Регистрация | Все |
| GET | `/user/{id}` | Получение пользователя | Все |
| GET | `/user/` | Список пользователей | Только auth |
| PATCH | `/user/{id}` | Обновление пользователя | Только свой или admin |
| DELETE | `/user/{id}` | Удаление пользователя | Только свой или admin |

### Объявления
| Метод | Эндпоинт | Описание | Доступ |
|-------|----------|----------|--------|
| POST | `/advertisement` | Создание объявления | Только auth |
| GET | `/advertisement/{id}` | Получение объявления | Все |
| GET | `/advertisement/` | Поиск с пагинацией | Все |
| PATCH | `/advertisement/{id}` | Обновление объявления | Только свой или admin |
| DELETE | `/advertisement/{id}` | Удаление объявления | Только свой или admin |
